# Prospect Agent — Licenças

Este repositório hospeda a lista de licenças revogadas do
software Prospect Agent (vendido pela Arkkhe).

## Como funciona

O arquivo `revoked.txt` contém os e-mails dos clientes cujas
licenças foram canceladas (reembolso, fraude, etc).

O aplicativo Prospect Agent consulta esta lista a cada 24h
e bloqueia o acesso quando o e-mail do usuário aparece aqui.

## Adicionando um e-mail

1. Edite `revoked.txt`
2. Adicione 1 e-mail por linha
3. Commit e push
